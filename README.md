# PWA Countdown Companion — Таймер очікування з підтримкою та офлайн-режимом

Інтерактивний мобільний веб-додаток (Progressive Web App), розроблений як персональний таймер зворотного відліку. Додаток створено для підтримки близьких людей під час тривалої розлуки. 

Головна технічна особливість проекту — **стовідсоткова автономність (Offline-first)**. Завдяки настроєному Service Worker, після першого запуску додаток разом із усіма медіафайлами (66 зображень підтримки та 10 святкових фото) зберігається в пам'яті телефону та працює без доступу до інтернету.

## 🚀 Функціонал додатку
* **Динамічний зворотний відлік:** Точний розрахунок днів, годин, хвилин та секунд до обраної дати у реальному часі.
* **Щоденний конвеєр підтримки:** Кожного дня додаток показує нову випадкову мотиваційну цитату (база з 66 цитат відомих психологів та мислителів) та нове миле зображення (дедуплікація вибору через `localStorage`).
* **Інтерактивні сценарії:** Робота з логічними екранами-перевірками стану користувача.
* **Екран особливого дня (День Народження):** Додаток автоматично розпізнає дату Дня народження користувача, зупиняє таймер та вмикає особливий інтерактивний привітальний екран з архівними фотокартками та листом підтримки.
* **Повністю адаптивний PWA-інтерфейс:** Оптимізовано під мобільні екрани, реалізовано плавні CSS-анімації переходу та кастомний екран інструкції з інсталяції додатку на Android/iOS.

## 🛠️ Стек технологій
* **Frontend:** Чистий HTML5, CSS3 (Flexbox, Grid, Glassmorphism-ефекти, адаптивна верстка).
* **Логіка додатку:** Чистий JavaScript (ES6+, робота з датами, `localStorage` для збереження стану та рандомізації медіафайлів).
* **PWA & Offline:** Service Worker API, Cache Storage API, веб-маніфест (`manifest.json`).

## 📲 Офлайн-архітектура (Service Worker)
Додаток використовує стратегію **Cache First (Спочатку кеш)** для миттєвого завантаження інтерфейсу та медіа-бази за відсутності мережі:

```javascript
self.addEventListener('fetch', (e) => {
    e.respondWith(
        caches.match(e.request).then((cachedResponse) => {
            return cachedResponse || fetch(e.request);
        })
    );
});
```
# ⚙️ Як кастомізувати проект під себе

Проект спроектовано так, щоб його було легко адаптувати для вашої близької людини:

1. Зображення: Замініть іконки у assets/icons/ та картинки підтримки у assets/images/ на власні фотографії.

2. Цитати: У файлі script.js відредагуйте масив initialQuotes, вписавши туди ваші персональні слова підтримки або улюблені фрази.

3. Особлива дата (День народження): У файлі script.js у функції updateTimer() змініть перевірку дати (наприклад, day === 1 && month === 1) на день народження вашої близької людини, щоб у цей день увімкнувся святковий екран.
